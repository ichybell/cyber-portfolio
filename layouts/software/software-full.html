SmartPotato: LSTM-Driven Market Intelligence

An Irish Potato Price Prediction Model bridging the digital divide by connecting high-frequency trading algorithms to low-tech USSD interfaces for rural farmers.

Operational Challenge

Smallholder farmers in volatile economies often sell their produce at a loss due to a lack of real-time market data. While sophisticated pricing models exist, they typically require high-bandwidth internet. The challenge was to predict potato prices accurately using time-series forecasting and deliver insights instantly to non-smartphones via USSD.

model_training.py 
def create_lstm_model(input_shape):
    model = Sequential()
    # LSTM Layer for Time Series
    model.add(LSTM(
        units=50, 
        return_sequences=True, 
        input_shape=(input_shape[1], 1)
    ))
    # Prevent Overfitting
    model.add(Dropout(0.2))
    model.add(LSTM(units=50))
    model.add(Dense(units=1)) # Price Output
    model.compile(loss='mean_squared_error', optimizer='adam')
    return model

Tech Stack: React.js/TypeScript, Flask API, PostgreSQL, Docker on GCP, with Africa's Talking USSD integration for offline access.

Machine Learning Implementation: LSTM

Data Engineering Strategy

  • filter_alt
    Winsorization: Applied to handle extreme price outliers caused by supply shocks, capping data at the 1st and 99th percentiles.
  • straighten
    MinMax Scaling: Normalized all pricing features to a [0, 1] range to ensure faster convergence of the LSTM gradient descent.

Offline Accessibility

To reach the target demographic of rural farmers, we integrated the Africa's Talking API. This allows users to query the model via simple USSD codes on feature phones.
